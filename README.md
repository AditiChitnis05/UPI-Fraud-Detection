# Realtime UPI Fraud Detection

This project detects fraudulent UPI transactions using machine learning.

## Features
- Fraud detection using ML model
- User & merchant interface
- Transaction monitoring

## Tech Stack
- Python
- Flask
- Machine Learning

## How to Run
1. Install dependencies
2. Run `app.py`
3. Open browser

- dataset/
