# Realtime UPI Fraud Detection

## Overview

This project detects fraudulent UPI transactions using machine learning and a web-based interface.

---

## Features

* Real-time fraud detection
* Machine learning model (CNN)
* User and merchant interface
* Transaction monitoring

---

## Tech Stack

* Python
* Flask
* TensorFlow / Keras
* HTML, CSS, JavaScript

---

## Project Structure

src/ – application code
models/ – trained models
dataset/ – data
templates/ – HTML
static/ – CSS/JS
docs/ – reports

---

## How to Run

```bash
git clone https://github.com/AditiChitnis05/UPI-Fraud-Detection.git
cd UPI-Fraud-Detection
pip install -r requirements.txt
python src/app.py
```

Open:
http://127.0.0.1:5000
